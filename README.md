# TechStore — Интернет-магазин профессионального инструмента

Современный веб-сервис для продажи инструментов (молотки, шуруповерты и др.) с мощной бэкенд-архитектурой и асинхронной обработкой задач.

## 🚀 Технологический стек

Проект построен на актуальном стеке технологий, который используется в крупных IT-компаниях:

* **Backend:** Python 3.10+ / Django 5.0
* **API:** Django Rest Framework (DRF)
* **Database:** PostgreSQL (Production-ready база данных)
* **Asynchronous Tasks:** Celery + Redis (асинхронная обработка заказов и фоновые задачи)
* **API Documentation:** Swagger / OpenAPI 3 (через drf-spectacular)
* **DevOps:** Docker / Docker Compose (полная контейнеризация всех сервисов)
* **Version Control:** Git

## ✨ Ключевые возможности

1.  **Каталог и поиск:** Умная фильтрация товаров по названию и описанию через Django QuerySets.
2.  **Система пользователей:** Полноценная регистрация и авторизация. Заказы привязаны к конкретным аккаунтам.
3.  **Асинхронность:** Оформление заказа происходит в фоновом режиме через Celery, что обеспечивает мгновенный отклик интерфейса.
4.  **Автоматизация:** Написана специальная Management Command (`seed_data`) для мгновенного наполнения базы тестовыми товарами.
5.  **Документированный API:** Все эндпоинты описаны в интерактивном интерфейсе Swagger.
